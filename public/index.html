Request/Response formats

  1. Person (omit id from POST and PUT methods)

    {
    ‘id’: number,
    ‘email’: String,
    'phones': [phone,phone,...](2),
    ‘firstname’: String,
    ‘lastname’: String,
    ‘address’: address(3),
    ‘hobbies’: [hobby, hobby, …](4),
    ‘zip’: number
    }

  2. Phone

    {
    'number': number,
    'description': String
    }

  3. Address

    {
    'street': String,
    'info': String
    }

  4. Hobby (omit everything but id for PUT and POST methods)

    {
    'id' number,
    'name': String,
    'link': string
    }
